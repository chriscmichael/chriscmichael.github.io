Coffee Bean Corral 2.0

  • Framework: Django
  • Backend: Python
  • Frontend: HTML/CSS
  • Database: SQLite

Coffee Bean Corral 2.0, my main passion project, is an innovative coffee tracking application. It provides a comprehensive record of coffee purchases, roasts, and brews, all made at home. The app boasts a robust backend powered by Django and Python, and a user-friendly frontend designed using HTML/CSS. The styling is elegantly executed without the use of Bootstrap, and the dropdown menus are implemented using Django choices for a seamless user experience.The first version of this app was written in Java during my time at Claim Academy. I wanted to see if I could do the same thing with Python after a few projects at Hack Reactor.

Completed November 2022
