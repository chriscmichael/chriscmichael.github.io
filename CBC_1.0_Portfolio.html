Coffee Bean Corral 1.0

  • Framework: Spring Boot
  • Backend: Java
  • Frontend: React
  • Database: MySQL

This was my capstone project for Claim Academy. This webpage tracks the green coffee beans I purchase, allows for searchability, signup, and login/logout functionality. CRUD functionality with both users and beans from the database. The first app that I wrote solo. I utilized ApacheTomcat, Maven, and Spring Boot for frameworks and the backend was written in Java. The frontend utilizes React JS for a Single Page Application. I wrote the HTML and CSS natively without Bootstrap.

Completed June 2022
