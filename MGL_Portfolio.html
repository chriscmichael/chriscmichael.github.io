MyGamesList

  • Framework: FastAPI
  • Backend: Python
  • Frontend: JavaScript/React
  • Database: MongoDB

The FastAPI and MongoDB-powered "MyGamesList" website was the capstone project for my final module at Hack Reactor. As part of a team of four, I contributed to the development of this web application by working on a separate Git branch and collaborating with my peers to create a final deployment branch. The site was my first project using FastAPI, a modern API framework with SwaggerUI for API calls, as well as my first experience with MongoDB, a NoSQL document-oriented database. My team implemented a Python-based API scraper that caches data on startup. The site was styled using Bootstrap and native CSS, providing an aesthetically pleasing and user-friendly interface for tracking the games played by users and their friends.

Completed January 2023
