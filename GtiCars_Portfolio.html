Git Cars or Fail Trying

  • Framework: Django
  • Backend: Python
  • Frontend: JavaScript/React
  • Database: PostgreSQL

Git Cars or Fail Trying, developed in a pair as the module two final project for Hack Reactor, is a web application designed for car dealerships to manage their inventory, sales, and service. Utilizing a powerful combination of Django, Python, React, and JavaScript, this website offers a seamless and intuitive experience for tracking all aspects of car dealership operations. Git Cars or Fail Trying uses RESTful API's that employ advanced polling techniques for API calls between inventory, sales, and service. This project was a major milestone for me, as it marked my first time utilizing Bootstrap to build active dropdown menus and form modals, further enhancing the user experience. The website utilizes separate Docker containers for the frontend, backend, and poller, demonstrating our ability to effectively manage complex projects.

Completed December 2022
