Project Tracker

  • Framework: Django
  • Backend: Python
  • Frontend: HTML
  • Database: SQLite

This was the first module final project at Hack Reactor. This was a solo project where I utilized my skills in Django to create a comprehensive web application that effectively tracks both projects and tasks. This project marked an important milestone in my development journey, as it was the first time I implemented essential features such as Login/Logout and Sign Up functionality. The project was organized as a root Django Project with task and project management through individual Django Apps, demonstrating my effective organization and management of large-scale projects.

Completed October 2022
